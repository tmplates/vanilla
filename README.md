# @tmplates/vanilla

VanillaJS project template config file.

This is a template configuration file that needs to be used with the [genrate](https://github.com/2046/genrate) tool.

## Preset

- Husky
- Babel
- ESLint
- Prettier
- CommitLint
- Jest [library]
- Rollup [library]
- TypeScript [library]
- Gulp [business]
- Sass [business]
- Postcss [business]
- BrowserSync [business]

## Usage

```bash
gen use @tmplates/vanilla [directory-name]
```

or

```bash
mkdir <directory-name>
cd <directory-name>
gen use @tmplates/vanilla
```
